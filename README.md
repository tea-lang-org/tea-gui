# tea-gui

## Navigate to the /GUI/tea_gui directory 
## GUI setup: Run this command on installation to install all necessary node modules 
```
npm install
```

### To run the server, run the following command 
```
npm run serve
```

### For production: 
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
